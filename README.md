# MacOSX-Kits

个人在OS X终端下的配置或小工具。

## USAGE

#### Installation

```
git clone --recursive https://github.com/JinnLynn/MacOSX-Kits.git && cd MacOSX-Kits && ./setup
```

#### Update

```
cd ~/.kits && git pull && git submodule update --init && ./setup
```

#### Misc

绝大部分的脚本功能都可以通过别名访问，具体查看`shell\alias.sh`中的注释。
